# WEB_IOT
The project takes information from the monitoring stations with MQTT protocols and render data for users
## Settings:
1. **Clone repository**:
   
      ```bash
   git clone https://github.com/hieu2711/WEB_IOT
2. **Set up**:
   - Client:
     ```sh
     cd client
     npm install
      ```
   - Server:
     ```sh
     cd client
     npm install
      ```
